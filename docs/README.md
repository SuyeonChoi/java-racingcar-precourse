# 자동차 경주 게임

## 📝 구현 기능 목록

- 경주할 자동차의 이름 입력받기
  - **[예외]** 이름이 5자 초과되는 경우
  - **[예외]** 중복된 이름이 입력되는 경우
    
- 경주를 시도할 회수 입력받기
  - **[예외]** 숫자가 아닌 문자를 입력하는 경우
  
- 경주 게임 수행
  - 입력된 경주 시도 회수 만큼 수행
  - 경주에 참여하는 모든 자동차에 대해 0에서 9 사이에서 무작위 값 생성
  - 생성된 값이 4이상인 경우 자동차 한 칸 전진 
  - 경주 차수별 모든 자동차의 위치 상태 출력

- 우승자 출력

- 오류 메세지 출력
